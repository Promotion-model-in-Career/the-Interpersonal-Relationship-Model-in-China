中式职场模型大致有6种人格特质，其中4种为底层人格特质，2种为中层人格特质。

![image](https://user-images.githubusercontent.com/17419617/142199545-c2870e70-458b-413c-b971-9fa2d7f75d3a.png)

基本属性可以分为打压（消极）与帮助（积极）两个大类。

    底层包括ABCD四种基本人格属性。

Type A

愚昧：

    不明事理
    
    直接获取物质利益
    
Type B

内斗：

    胡搅蛮缠
    
    恣意争夺扰乱他人

Type C

绕弯：

    缓慢办事
    
    缓慢办事实际处理

Type D

装逼：

    技能升华
    
    自己利益最为重要
